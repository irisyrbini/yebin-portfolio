## 파일명 규칙

- 대소문자까지 정확히 맞춰주세요.
- 공백 대신 하이픈 `-`을 사용하세요.
- 이미지는 `.jpg` 기준으로 연결되어 있습니다.
- 영상은 `.mp4` 기준으로 연결되어 있습니다.
- 다른 확장자를 쓰려면 `index.html`의 `src`도 같이 바꿔야 합니다.

| `img/personal/ttc_romanticized.jpg` | TTC romanticized — 인터루드 풀뷰포트 배경 (Pinterest 바이럴 핀) | 1600×2000px 세로 권장 |
