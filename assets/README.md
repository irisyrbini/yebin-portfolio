# Yebin Portfolio Assets Checklist

이 폴더 안에 아래 이름 그대로 파일을 넣으면 `index.html`에서 자동으로 표시됩니다.
파일명이 다르면 화면에는 placeholder가 계속 보입니다.

원칙:
- 대소문자까지 정확히 맞춰주세요.
- 공백 대신 하이픈 `-`을 사용하세요.
- 이미지는 `.jpg`, 영상은 `.mp4`, 문서는 `.pdf` 기준입니다.
- 같은 파일 구조를 PM 포트폴리오에도 맞춰두면 두 사이트에 동시에 재사용하기 쉽습니다.
